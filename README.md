🏥 Healing Touch Hospital Patient Registry System

A simple and efficient Hospital Patient Registry System built using Node.js, Express.js, HTML, and CSS. This application enables hospital staff to register patients, store patient information, and view records through an easy-to-use web interface.

📖 Overview

The Healing Touch Hospital Patient Registry System is designed to simplify patient record management. Users can enter patient details such as name, date of birth, and disease/illness through a registration form. The application processes and stores the information in a text file, allowing staff to access patient records whenever needed.

✨ Features
📝 Patient registration form
📂 Store patient details in a file
👨‍⚕️ View all registered patient records
🎨 Modern and responsive user interface
⚡ Fast backend powered by Express.js
💾 Lightweight file-based data storage
